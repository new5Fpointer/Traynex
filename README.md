# Traynex

## 简介

Traynex 是一款 Windows 桌面工具，可将任意应用窗口最小化到系统托盘（又称“通知区域”）。该项目受到 [Traynard](https://github.com/tabris17/traynard) 启发，并使用 [QT](https://www.qt.io/zh-cn/product/framework) 完全重写。

## 功能

1. 全局热键
2. 多语言支持
3. 高级功能

## 安装

### 1. 下载
- 从 [Releases](https://github.com/new5fpointer/Traynex/releases/latest) 页面下载压缩包
- 从 [蓝奏云](https://new-pointer.lanzouu.com/b00yap47fc?pwd=7gin) 下载版本号最新的压缩包
   
### 2. 安装
1. 在本地解压
2. 运行 **traynex.exe** 可执行文件
   
## 用法

### 隐藏窗口
1. **通过热键隐藏**：选中目标窗口，按下设置的热键（默认 `Win + Shift + Z`）隐藏到托盘图标
2. **通过右键菜单隐藏**：
   - 在主窗口列表中选择窗口，右键点击"隐藏到托盘图标"或"隐藏到托盘菜单"
     - "隐藏到托盘图标"：窗口会显示为独立的系统托盘图标
     - "隐藏到托盘菜单"：窗口会添加到 Traynex 的托盘菜单中
  
### 恢复窗口

1. **选择恢复**
    - 双击系统托盘图标或在托盘菜单中点击
    - 在"隐藏窗口"页面中选择恢复

2. **快捷恢复**
   - 恢复所有：使用"恢复所有窗口"功能
   - 恢复上一个：使用"还原上一个窗口"功能

**所有恢复操作都可以通过系统托盘完成**

### 高级右键功能
- **前置窗口**：快速将后台窗口带到前台
- **高亮窗口**：在多个窗口中快速定位目标窗口
- **窗口置顶**：让重要窗口始终显示在最前面
- **结束任务**：强制关闭无响应的窗口进程

### 系统托盘操作
- **左键单击**：显示/隐藏主窗口
- **右键点击**：显示功能菜单
  - **打开主窗口**
  - **隐藏到托盘的窗口列表**（若没有则不显示）
  - **还原上一个窗口**
  - **恢复所有窗口**
  - **退出**
   
## 设置说明
### 热键设置

- 点击"设置热键"按钮，按下想要的组合键

  - 支持修饰键：Ctrl, Alt, Shift, Win

  - 支持功能键：F1-F12，字母键，数字键等

### 常规设置
1. 开机自启
   - 控制是否随系统启动

2. 启用热键
   - 启用或禁用全局热键功能

3. 始终置顶
   - 主窗口是否保持在最前端

4. 语言设置
   - 切换界面语言
     1. 中文
     2. English

### 刷新设置
- 自动刷新：启用窗口列表自动刷新

- 刷新间隔：设置刷新频率（100-1000毫秒）
